[003] 2025 · advanced engineering informatics
generating command modeling and design graphs with data augmentation for enhanced 3d modeling support
yugyeong jang · kyung hoon hyun

this study proposes a system that automatically generates 3d modeling sequences for various 3d shapes. we developed a method for inferring and extracting modeling sequences from 3d shapes to generate command modeling and design graphs without the need for additional modeling data collection. the proposed system infers modeling sequences from completed 3d models and converts them into workflow graphs, providing richer and more detailed sequence data compared to existing datasets.

[002] 2025 · archives of design research
BIGcad: assisting 3d cad modeling with workflow graph-driven bayesian command inference
kyung hoon hyun · yugyeong jang

designers continue to rely on command-based cad tools like rhino for precise and complex 3d modeling. to address this, we propose a framework combining a bayesian inference model and workflow graph to predict user intentions, suggesting tailored commands and sequences to improve precision and efficiency. the framework was developed as a rhino plugin. user studies show our system reduces errors, streamlines workflows, and offers effective command recommendations.

[paper]
[001] 2024 · ACM CHI 2024 LBW
advancing 3d cad with workflow graph-driven bayesian command inferences
yugyeong jang · kyung hoon hyun

advancements in 3d generative ai have significantly improved design capabilities. however, the focus on mesh-based models limits opportunities for detailed modifications. addressing this, we propose a system that predicts user modeling steps using an inference model based on behavior, thereby promoting efficient workflow and precise command usage. user studies demonstrate that our system minimizes modeling errors, streamlines processes, and offers recommendations for effective command usage.
